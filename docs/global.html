remove_allow_empty_children(identifier, indiscriminate)

Function removes children of the node identified by the identifier argument, and the node itself if it has the class .allow-empty. If children are matched then nothing happens, as no nodes are allowed to be emptied. When set to true, the indiscriminate variable means that when nodes or their children have the class .allow-empty, it will remove it's descendents. If the indiscriminate variable is false, then it will only remove descendents that in turn do not have descendents with the .allow-empty class.

indiscriminate * true
